Tool tips

This 1st tooltip has been setup with a preference to show below the opening link if there is enough space available. This preference can be specified in the plugin options - $("#tooltip1").tooltip({ position: 'bottom' });. If no preference is supplied the plugin will try to position the tooltip on top. If there is not sufficient space in the preferred place it will try to position in the most appropriate place. Open tooltip (small) - position bottom
