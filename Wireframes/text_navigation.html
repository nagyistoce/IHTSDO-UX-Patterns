TEXT NAVIGATION

IHTSDO

INTERACTION

Used as primary navigation for leading users within a service.

WHAT IS IT FOR?

Labels are stacked to create a navigation list and used on-page to guide a user between features of a service. Navigation items have both a default and rollover state that should be observed consistently. Each label has a default, rollover and selected state. These differ for lists on a white background and on a dark background, but should be observed consistently in their use across all services.

BEHAVIOUR

For a contained and consistent arrangement, labels should not exceed the width of the column they are featured in, so avoid lengthy labels. Commonly, the width should be 4 columns (of 16 column grid).

 

Subsections of labels within the same list can be divided by a thin line.
