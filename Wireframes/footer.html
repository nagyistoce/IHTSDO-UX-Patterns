FOOTER

IHTSDO

INTERACTION

Supporting the role of the masthead, the footer can contain expanded top-level functionality. It also provides a logical base to a page. It can contain a broader set of less primary or extended functions and links or static text. These may include legal and policy information, support or external links.

WHAT IS IT FOR?

The footer spans the whole screen width and exists beneath the main screen content. It is exposed when the end of a screen is reached. It responds in a liquid manner to changes in browser width.

BEHAVIOUR

To maintain a consistent experience, the footer should be present on all pages at all times. Together with the masthead, it also serves to 'bookend' page content.
