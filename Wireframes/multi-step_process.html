To guide a user through a linear task flow with clearly defined, sequential steps. Multi-step indicators facilitate in communicating how many steps are involved and progress accordingly.

WHAT IS IT FOR?

MULTI-STEP PROCESS INDICATOR

IHTSDO

INTERACTION

Each instance of the indicator changes state (colour) when a step is completed. This is usually on-click of a submit or action button at the end of a step. Steps to be completed remain their default state.

BEHAVIOUR

The multi-step process indicator should be implemented in cases when a user is required to complete a task in a defined order. Be mindful of how many steps to include. Ideally, steps should not exceed a maximum of six. Any more will lower a user's willingness to complete.

 

Note: the step process indicator is NOT to be used as a breadcrumb.

HOW AND WHEN TO USE

EXAMPLES

Do not use when there are only one or two steps as part of a task or flow.
