Radio Buttons

Radio buttons allow the choice of a single element from a pool of elements.

 

Checkboxes

Checkboxes allow for the user to select multiple elements.

WHAT ARE THEY FOR?

RADIOS & CHECKBOXES

IHTSDO

INTERACTION

Radio Buttons

When a radio button is activated, all other radio buttons are deactivated. This parameter is parsed as part of a form or option selection.

 

Checkboxes

The user can select (switch on/off) arbitrary elements from a small group of elements that do not cancel each other out. A single checkbox can be used to toggle directly opposing states. e.g. 'Yes/No' or 'Agree/Disagree'.

BEHAVIOUR

Radio Buttons

When only one selection is possible. Radio buttons are only pre-selected when the selected option is the most likely option that will be chosen by users and the selected choice cannot be undone.

 

Checkboxes

When multiple options can be selected. This could be as part of a results set filtering system, where multiple selection reduce the set. Checkboxes can be default all on or all off.
