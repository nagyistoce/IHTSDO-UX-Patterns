Steppers are a UI element that enable a user to set a preferred quantity as part of a form, product or allocation method that requires an amount to be specified.

WHAT IS IT FOR?

STEPPER

IHTSDO

INTERACTION

On-click of -/+ control icons, the counter display increases or decreases the unit amount by one digit accordingly. It's also possible to focus in the field and enter a numerical amount manually. Observe the field on-focus highlight behaviour (see forms for details). Similarly, the field feeds back interaction with the -/+ buttons with the same highlight.

BEHAVIOUR

The stepper default unit amount can be defined dependent upon context. In selecting a product, a default of '1' would be recommended in order to mitigate against errors. However, when assigning tasks as part of mapping, a default of '0' (zero) would be suggested, to ensure user makes their exact selection.

 

When used as part of a form or assigning process, the amount is parsed on-click of a submit/action button.
