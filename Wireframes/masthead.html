MASTHEAD

IHTSDO

INTERACTION

The masthead forms an anchor from which all screen and interface content hangs. It's used to hold important information and interactions relating to the active screen at that time. This typically includes the brand mark, header, sub header and interactive assets.

WHAT IS IT FOR?

The masthead spans the whole screen width and is used to nest control functions and features at the highest level. Examples include username, logout, home, tools, or dashboard, but always the brand mark.

 

As screen content is scrolled, the masthead locks, allowing content to flow beneath it. This means it is always visible at the top of the screen. It responds in a liquid manner to changes in browser width.

BEHAVIOUR

To maintain a consistent experience, the masthead should be present on all pages at all times. It’s important that it is applied accurately across all IHTSDO Services, to further support a consistent brand experience every time.

 

To clearly identify a currently viewed IHTSDO Service, a 'locked off' text header system is used together with the brand mark. A consistent type size should be used across all services.
