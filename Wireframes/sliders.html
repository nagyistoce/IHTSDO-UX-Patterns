Unit 1

 

Sliders are a UI input element allowing selection of a point or range on a numeric scale.

WHAT ARE THEY FOR?

SLIDERS

IHTSDO

INTERACTION

 

Single Value Slider

User drags slider handle horizontally up/down the range. Unit is fed back as text display. Increment feedback is defined by requirement. It could be a continuous feedback, or on wider scale points.

 

Range Slider

User drags slider handles independently to select the required range. Range is defined numerically on each handle.

BEHAVIOUR

Sliders are often used to reduce user effort and input as they have only to drag the slider until the required value is met, avoiding any keyboard interaction to specify a value. They are used in cases of selecting a value.
