Accordions take a large amount of information or data and encapsulates it in a smaller area. Typically, they are used as part of a set of like-content, but allows the user to focus on one at a time.

WHAT ARE THEY FOR?

ACCORDIONS

IHTSDO

INTERACTION

Physically, they are blocks organised in a vertical manner, such that once a tab is clicked, a pane slides out to reveal and focus a user's attention to the further content to be explored in more depth. Consequently, neighbouring modules are pushed downward. Also, each header uses an icon to symbolise whether the accordion tab is opened or not. The icon is also clickable.

BEHAVIOUR

Accordions should be used when a user needs to browse a large results set and wishes to explore each in more depth, without navigating away to another page.

 

Headers of each tab should be large enough to make them easily clickable. They also house the label that should clearly communicate the content within.

 

Accordion pane content can be of a variety: text, image, diagram, video, tabs. The latter is an example that is implemented for SNOMED CT browsing of each term's associated details etc.
