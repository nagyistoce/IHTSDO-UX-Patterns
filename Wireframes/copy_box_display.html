COPY BOX DISPLAY

INTERACTION

The box has its own style that helps to distinguishes it from the rest of the page. It should allow a user to select all or part of the copy as required.

BEHAVIOUR

The copy box display area should be used when you wish to present information that the user needs to copy and later paste into another application.
